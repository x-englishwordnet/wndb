# Extended English Wordnet in WNDB format

### new

Some new types of relations are introduced with ad-hoc encoding that may not be recognized by some libraries (JWI) while others don't bother (NLTK).
- the new 'is caused by' relation is encoded with '&gt;&circ;'
- the new 'is entailed by' relation is encoded with '*&circ;'

This is compatible with PWN3.1 format and may be required to work with some libraries (e.g. JWI)

This provides ZIP or TAR.GZ files of the Extended WordNet files in [WNDB](https://wordnet.princeton.edu/documentation/wndb5wn) format.

Downloadable [here](https://x-englishwordnet.github.io/wndb/xewn.dict.tar.gz) :
